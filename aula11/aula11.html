Diagnsticos da anlise de varincia

Todo delineamento experimental possui um modelo matemtico que o representa, e deve ser levado em conta na anlise de varincia, aceitando algumas hipteses bsicas necessrias para a validade da anlise.

No caso do \(DIC\), por exemplo, o modelo matemtico definido como:

\[ y_{ij}= \mu + \tau_i + \epsilon_{ij} \] onde:

\(y_{ij}\): o valor observado na parcela que recebeu o tratamento i na repetio j;

\(\mu\): a mdia geral do experimento;

\(\tau_i\) o efeito devido ao tratamento i que foi aplicado parcela;

\(\epsilon_{ij}\) o efeito dos fatores no controlados na parcela que recebeu o tratamento i na repetio j.

Hipteses bsicas para aplicao da anlise de varincia

As hipteses bsicas que devemos admitir para tornar vlida a aplicao da anlise de varincia so as seguintes:

  1. Aditividade: Os efeitos dos fatores do modelo so aditivos.

  2. Independncia: Os erros (desvios) \(\epsilon_{ij}\), devido aos efeitos de fatores no controlados ou acaso devem ser independentes.

  3. Homocedasticidade (homogeneidade de varincias): Os erros (desvios) \(\epsilon_{ij}\), devido ao acaso devem possuir varincia comum \(\sigma^2\)

  4. Normalidade dos desvios: Os erros ou desvios \(\epsilon_{ij}\) devido ao acaso devem possuir distribuio normal de probabilidades.

Uma forma resumida de apresentar estas quatro pressuposies apresentada a seguir:

\[ \epsilon_{ij} \overset{iid}{\tilde{} }N(0,\sigma^2) \\ \]

Quando as hipteses bsicas no so satisfeitas, como no caso mais frequente a homocedasticidade, deve-se utilizar uma transformao dos dados para contormar o problema.

Transformaes

1. Raiz Quadrada: \(y'=\sqrt{y}\)

Geralmente utilizada para dados de contagem, que frequentemente seguem a distribuio de Poisson, em que a mdia e a varincia so iguais. Exemplo, nmero de caros por folha, nmero de frutos atacados, nmero de plantas doentes por parcela, etc. No caso de ocorrncia de valores nulos ou baixos, recomenda-se \(y'=\sqrt{y+0,5}\) ou \(y'=\sqrt{y+1}\).

2. Arco Seno: \(y'=arcoseno\sqrt{\frac{y}{100}}\)

Recomendada para dados de porcentagem, proveniente de contagem, que geralmente seguem a distribuio binomial, como exemplo, \(\%\) de plantas atacadas, \(\%\) de germinao, etc.

3. Tranformao Potncia

Quando constatada uma certa proporcionalidade entre as mdias e os desvios padres dos tratamentos. Exemplo: No caso de contagem de insetos, se a populao numerosa, as contagens sero altas para a testemunha e para os tratamentos que no controlam a praga, enquanto que, para os tratamentos que controlam a praga, a amplitude de variao ser baixa.

Pode-se verificar isso pelo diagrama de disperso, visualizando a reta \(y = a + b x\), plotando-se: \(x = log(mdia)\) versus \(y = log(desvio padro)\).

  • Se \(b=2,0\), a transformao adequada a recproca: \(y' = y^{-1}\);
  • Se \(b=1,5\), a transformao adequada a recproca da raiz quadrada: \(y' = y^{-1/2}\);
  • Se \(b=1,0\), a transformao adequada a logartmica : \(y' =log(y)\);
  • Se \(b=0,5\), a transformao adequada a raiz quadrada : \(y' = y^{1/2}\).
  • Se \(b=0\), no faa transformao.

4. Tranformao Box-Cox

A transformao de Box-Cox dada pela seguinte frmula:

\[ W_i = Y_i^\lambda, \text{ se } \lambda \neq 0 \\ W_i = Ln(Y_i), \text{ se } \lambda = 0 \\ W_i = Y_i, \text{ se } \lambda = 1 \] onde \(Y_i\) um valor de dados original e \(\lambda\) o parmetro para a transformao. Observe que \(\lambda = 1.0\) NO TRANSFORMAR.

\[ \begin{aligned} \lambda &= -1.0, \text{ } x_i(\lambda) = \frac{1}{x_i} \\ \lambda &= -0.5, \text{ } x_i(\lambda) = \frac{1}{\sqrt{x_i}} \\ \lambda &= 0.0, \text{ } x_i(\lambda) = Ln(x_i) \\ \lambda &= 0.5, \text{ } x_i(\lambda) = \sqrt{x_i} \\ \lambda &= 2.0, \text{ } x_i(\lambda) = x_i^2 \end{aligned} \] A utilizao da transformao Box-Cox, por ser uma transformao do tipo potncia, no detecta alguns tipos de transformao classicamente utilizados, por exemplo arco seno raiz(porcentagem/100), usado para porcentagens binomiais.

mod <- lm(resp ~ tempo,
          data = dados)
boxcox(mod)

mod2 <- lm(log(resp) ~ tempo,
          data = dados)
boxcox(mod2)
